## Anti-Patterns Verdict
Above average, but scaffolding gives it away. Real identity (WebGL trails, serif display, calm palette, anti-sell voice) undercut by (1) mono eyebrow + numbered markers on nearly every section, (2) uniform card hover choreography. Detector (4 findings): numbered-section-markers (advisory), aphoristic-cadence x4 (warning), em-dash-overuse x10 (warning), layout-transition nav width (warning, line 214). No false positives.

## Priority Issues
- [P1] Scroll-reveal gates visibility + ignores prefers-reduced-motion (blank-section risk, WCAG AA violation). Detector can't see it. Fix: visible-by-default + reduced-motion fallback; drop slide-out. -> /impeccable animate
- [P2] Mono eyebrow + numbered 01..07 markers on every section. Fix: numbers only where order matters; quieter cadence elsewhere. -> /impeccable typeset
- [P2] Uniform hover (lift + accent line + shine sweep) on every card; competes with trails. Fix: reserve rich hover for one tier. -> /impeccable quieter
- [P2] Copy tells: 10 em-dashes + 4 aphorisms undercut human voice. Fix: convert dashes, rewrite 3 of 4 aphorisms. -> /impeccable clarify
- [P3] Nav underline animates width (line 214). Fix: transform: scaleX. -> /impeccable optimize

## Persona Red Flags
- Jordan (first-timer): reveal-gated section can appear empty on slow JS / reduced-motion.
- Casey (mobile): WebGL + autoplay videos heavy on 3G; footer link targets close.
- Marcus (skeptical coach, project persona): numbered scaffolding + synchronized hover read "agency template" — the exact skepticism the brand fights.

## Minor Observations
- Bi-directional reveal (animate out on scroll-up) is a uniform reflex.
- Hover shine sweeps animate `left`; prefer transform.
- audit.html + booking "Fill Intake Form" link now redundant with removed inline audit.
